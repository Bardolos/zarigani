Zarigani

Welcome to the introductory Zarigani page. You will find a brief description as well as an example of one of Zarigani's many uses.

What is Zarigani?

Literally speaking, Zarigani is a species of crayfish endemic to Japan. While this aplication is colored like one, and features a suggestive icon, it has nothing to do with seafood.

Zarigani (the application) is written in Ruby and its main purpose is to transform Markdown into Markup (namely, HTML) in real time, side by side.

This isn't really impressive, but Zarigani allows for more than that: you can write the <head> wrapper in a separate file, in which you're free to include styling and functionality - like CSS and Javascript, much like you would on a real web page. Zarigani works as a surrogate browser, allowing you to incorporate content from the web (like images, javascript files, etc.) into your Markdown, with the immediate result being visibl ein HTML.

The motivation is quick prototyping. Write the <head> - called a decorator in Zarigani - once with all the style and functionality, then write all the Markdown you need, and that wrapper will be automatically envelop it. Zarigani supports changing both the decorator's contents, as well as the specific decorator file dynamically - meaning you can use Zarigani and style your HTML with your favourite editor at the same time as you go. Simply press Refresh or start typing to apply the changes.

Zarigani is currently in advanced beta.
